# Bibelhenvisninger og bokforkortelser

Konvensjon for Åpen Bibel. Følger moderne norsk skikk (Bibelselskapet 2011/2024-tradisjon), ikke 1930-kodene (`1MO`, `SLM`, `JES` osv.).

## Format

```text
Sal 139,16.
1 Krøn 29,30; Sal 139,16.
Jes 40,13.
```

| Regel | Eksempel |
|-------|----------|
| Komma mellom kapittel og vers | `Sal 139,16` (ikke `139:16`) |
| Ingen mellomrom inne i `kapittel,vers` | `29,30` |
| Semikolon mellom flere steder | `1 Krøn 29,30; Sal 139,16.` |
| Mellomrom i flerdels-forkortelser | `1 Mos`, `1 Krøn`, `1 Kor` |
| Punktum til slutt i fotnote | `…16.` |
| Versrekke med tankestrek | `Rom 5,6–11` |

Fotnote i tekstfiler:

```markdown
[^1]:  1 Krøn 29,30; Sal 139,16.
```

(To mellomrom etter kolon — eksisterende konvensjon.)
